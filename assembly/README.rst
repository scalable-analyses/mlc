Assembly Language
=================

The ``data`` directory contains the functions ``inner_product_cpp`` and ``outer_product_cpp`` in the file ``base_math.cpp``. 
The function ``inner_product_cpp`` computes the inner product of two vectors.
The function ``outer_product_cpp`` computes the outer product of two vectors. 

.. admonition:: Tasks

   1) Implement the inner and outer products in AArch64 assembly and write unit tests. 
   2) Use the GNU Project Debugger `GDB <https://www.sourceware.org/gdb/>`__ to step through an example call to the inner product function.
